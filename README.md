# 📊 Zomato Data Analysis

## 📌 Overview
This project analyzes restaurant data to identify trends in cuisines, pricing, and customer ratings. The goal is to derive insights that can help understand customer preferences and business opportunities in the food industry.

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

---

## 📂 Dataset
The dataset contains restaurant details such as:
- Name  
- Location  
- Cuisines  
- Rating  
- Cost for two  

---

## ⚙️ Approach

1. Data Cleaning  
   - Removed missing values  
   - Converted ratings and cost into numeric format  

2. Data Analysis  
   - Identified top cuisines  
   - Analyzed rating distribution  
   - Compared price vs rating  

3. Visualization  
   - Bar chart for cuisines  
   - Histogram for ratings  
   - Scatter plot for price vs rating  

---

## 📊 Key Insights
- Certain cuisines dominate the restaurant market  
- Most ratings fall between 3.5–4.5  
- Higher price does not always guarantee better ratings  

---

## 📈 Output
- Top cuisines chart  
- Rating distribution  
- Price vs rating visualization  

---

## 🚀 Conclusion
This project demonstrates how data analysis can be used to extract insights from real-world datasets and support decision-making in the food industry.
